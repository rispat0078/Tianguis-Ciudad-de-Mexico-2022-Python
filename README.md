## 📄 About This Application

This application helps you explore and visualize data about tianguis (open markets) in Mexico City for the year 2022. It uses Python to analyze and map the locations and behavior of these markets throughout the week. The data comes from the Secretaría de Desarrollo Económico (SEDECO).

You do not need to know programming to use it. This guide will walk you step-by-step through downloading, setting up, and running the software on Windows.

---

## 🖥️ System Requirements

- Windows 10 or higher (64-bit recommended)  
- At least 4 GB of free disk space  
- Internet connection to download the software and updates  
- Python 3.7 or later (installation steps included)  
- Basic permissions to install software on your computer  

---

## ⚙️ What This Software Does

- Loads tianguis data from SEDECO for Mexico City in 2022  
- Performs exploratory data analysis (EDA) to reveal patterns  
- Creates maps showing tianguis locations across the city  
- Displays market activity changes by day of the week  
- Helps identify geographic trends and hotspots  

You will see charts and maps that make it easier to understand where and when these open markets operate.
